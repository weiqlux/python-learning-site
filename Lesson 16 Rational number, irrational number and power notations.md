# Lesson 16 Rational number, irrational number and power notations

# 第十六讲 有理数、无理数和幂

# 有理数和无理数

1.  有理数：如果一个数可以表示为$\frac{m}{n}$（m和n是整数，且n$\neq$`<!-- -->`{=html}0），那么这个数就是有理数。

2.  无理数：如果一个数不能表示为$\frac{m}{n}$（m和n是整数，且n$\neq$`<!-- -->`{=html}0），那么这个数就是无理数。或者说：无限不循环小数称为无理数。

# 平方根、算术平方根和算术基本运算

1.  "$\sqrt{}$"表示对一个数或代数式进行开方运算，若$a^{2} = n$，则$\sqrt{n} = a(a \geq 0)$。

2.  平方根：若$a^{2} = n$，则a是n的平方根。

3.  算术平方根：一个数（非负）的两个平方根中，非负的那个平方根也叫算术平方根。

4.  最简二次根式：被开方数不含字母；被开方数中不含能开得尽方的因数或因式；分母中不含根式，满足上述三个条件的二次根式，叫做最简二次根式。

5.  加减法：先把二次根式化简，然后把被开方数相同的二次根式（即同类二次根式）的系数相加减，被开方数不变。

6.  乘除法:

    两个代数式的算术平方根的积或商，等于两个代数式的积或商的算术平方根。

# 幂

1.  n个相同的数a相乘的运算，叫做乘方，记作$a^{n}$，乘方的结果叫做幂（power）。

2.  同底数幂法则：同底数幂相乘除，原来的底数作底数，指数的和或差作指数。

    $a^{m} \times a^{n} = a^{(m + n)}$ $a^{m} \div a^{n} = a^{(m - n)}$

3.  同指数幂法则：同指数幂相乘除，指数不变，底数相乘除。

    $a^{m} \times b^{m} = {(ab)}^{m}$
    $a^{m} \div b^{m} = {(\frac{a}{b})}^{m}$

4.  幂的乘方法则：幂的乘方，底数不变，指数相乘。

    $${(a^{m})}^{n} = a^{(m \times n)}$$

5.  积的乘方法则：积的乘方，先把积中的每一个因数分别乘方，再把所得的幂相乘。

    $${(a \times b)}^{m} = a^{m} \times b^{n}$$

6.  指数为0幂法则：$a^{0} = 1$，其中$a \neq 0$

7.  负整数指数幂法则：$a^{( - k)} = \frac{1}{a^{k}}$，其中$a \neq 0$，$k \in N^{\ast}$

## 例题

**例题1**：判断下列数哪些是有理数，哪些是无理数：$\frac{3}{4}$，$\sqrt{2}$，$0.5$，$\pi$，$-\frac{5}{7}$。

**解**：
- $\frac{3}{4}$：有理数（可以表示为分数）
- $\sqrt{2}$：无理数（无限不循环小数）
- $0.5$：有理数（可以表示为$\frac{1}{2}$）
- $\pi$：无理数（无限不循环小数）
- $-\frac{5}{7}$：有理数（可以表示为分数）

**例题2**：求 $|-5|$ 和 $|3-8|$ 的值。

**解**：$|-5| = 5$（负数的绝对值等于它的相反数）
$|3-8| = |-5| = 5$

**例题3**：求 $\sqrt{16}$ 和 $\sqrt{25}$ 的值。

**解**：$\sqrt{16} = 4$（因为 $4^2 = 16$）
$\sqrt{25} = 5$（因为 $5^2 = 25$）

**例题4**：计算 $2^3 \times 2^4$。

**解**：根据同底数幂相乘法则：
$2^3 \times 2^4 = 2^{3+4} = 2^7 = 128$

**例题5**：计算 $5^6 \div 5^3$。

**解**：根据同底数幂相除法则：
$5^6 \div 5^3 = 5^{6-3} = 5^3 = 125$

**例题6**：计算 $(2^3)^4$。

**解**：根据幂的乘方法则：
$(2^3)^4 = 2^{3 \times 4} = 2^{12} = 4096$

**例题7**：计算 $(3 \times 5)^2$。

**解**：根据积的乘方法则：
$(3 \times 5)^2 = 3^2 \times 5^2 = 9 \times 25 = 225$

**例题8**：计算 $2^{-3}$。

**解**：根据负整数指数幂法则：
$2^{-3} = \frac{1}{2^3} = \frac{1}{8}$

**例题9**：化简 $\sqrt{12}$。

**解**：$\sqrt{12} = \sqrt{4 \times 3} = \sqrt{4} \times \sqrt{3} = 2\sqrt{3}$

**例题10**：计算 $\sqrt{8} + \sqrt{18}$。

**解**：$\sqrt{8} = 2\sqrt{2}$，$\sqrt{18} = 3\sqrt{2}$
$\sqrt{8} + \sqrt{18} = 2\sqrt{2} + 3\sqrt{2} = 5\sqrt{2}$
