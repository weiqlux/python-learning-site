# Lesson 24 Proportional Distance Problem

# 第二十四讲 比例行程问题

1.  **Proportional relationships in journey problems
    行程问题中的比例关系（S = V × T）**

    速度一定，路程与时间成正比；

    时间一定，速度与路程成正比；

    路程一定，速度与时间成反比。

2.  **Use proportions to solve travel problems 比例解行程问题**

1.  当两个物体运行速度在所讨论的路线上保持不变时，经过同一段时间后，他们走过的路程之比就等于他们的速度之比。

    ![1754034895503](media/image4.png){width="1.5020833333333334in"
    height="1.0493055555555555in"}\
    因为时间相同，即$t_{甲}\  = \ t_{乙}\  = \ t$，所以由$t_{甲}\  = \ \frac{s_{甲}}{v_{甲}}$，
    $t_{乙}\  = \ \frac{s_{乙}}{v_{乙}}$，得$t\  = \ \frac{s_{甲}}{v_{甲}}\  = \ \frac{s_{甲}}{v_{甲}}$，
    $\frac{s_{甲}}{s_{乙}}\  = \ \frac{v_{甲}}{v_{乙}}$，甲乙在同一时间t内的路程之比等于速度比。

2.  当两个物体运行速度在所讨论的路线上保持不变时，走过相同的路程时，两个物体所用的时间之比等于他们速度的反比。

    ![](media/image5.png){width="1.2222222222222223in"
    height="0.8034722222222223in"}\
    因为路程相同，即$s_{甲}\  = \ s_{乙}\  = \ s$，所以由$s_{甲}\  = \ v_{甲}\  \times \ t_{甲}$，$s_{乙}\  = \ v_{乙}\  \times \ t_{乙}$，得$s\  = \ v_{甲}\  \times \ t_{甲}\  = v_{乙}\  \times \ t_{乙}\ \ $，$\frac{v_{甲}}{v_{乙}}\  = \ \frac{t_{甲}}{t_{乙}}$，

    甲乙在同一段路程$s$上的时间之比等于速度比的反比。

## 例题

**例题1**：甲、乙两人同时从A地出发到B地，甲的速度是每小时60千米，乙的速度是每小时40千米。如果两人同时到达B地，求甲、乙两人走过的路程比。

**解**：因为时间相同，路程比 = 速度比 = $60:40 = 3:2$

**例题2**：甲、乙两人从A地到B地，路程相同。甲的速度是每小时50千米，乙的速度是每小时40千米。求甲、乙两人所用时间的比。

**解**：因为路程相同，时间比 = 速度比的反比
速度比 = $50:40 = 5:4$
时间比 = $4:5$

**例题3**：甲、乙两人从同一起点出发，甲的速度是乙的2倍。如果两人同时出发，经过相同时间后，甲走了60千米，求乙走了多少千米。

**解**：因为时间相同，路程比 = 速度比 = $2:1$
设乙走了x千米，则：
$\frac{60}{x} = \frac{2}{1}$
$x = 30$ 千米

**例题4**：甲、乙两人要走相同的路程，甲的速度是每小时30千米，乙的速度是每小时20千米。如果甲用了2小时，求乙用了多长时间。

**解**：因为路程相同，时间比 = 速度比的反比
速度比 = $30:20 = 3:2$
时间比 = $2:3$
设乙用了t小时，则：
$\frac{2}{t} = \frac{2}{3}$
$t = 3$ 小时

**例题5**：一辆汽车以恒定速度行驶，前3小时行驶了180千米，后2小时行驶了120千米。求这辆汽车的平均速度。

**解**：总路程 = $180 + 120 = 300$ 千米
总时间 = $3 + 2 = 5$ 小时
平均速度 = $300 \div 5 = 60$ 千米/小时

**例题6**：甲、乙两人从A、B两地同时出发，相向而行。甲的速度是每小时50千米，乙的速度是每小时40千米。如果A、B两地相距270千米，求他们相遇时各自走过的路程。

**解**：因为时间相同，路程比 = 速度比 = $50:40 = 5:4$
设甲走了5x千米，乙走了4x千米，则：
$5x + 4x = 270$
$9x = 270$
$x = 30$
所以甲走了 $5 \times 30 = 150$ 千米，乙走了 $4 \times 30 = 120$ 千米
