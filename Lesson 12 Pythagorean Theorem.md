# 毕达哥拉斯定理

1.  毕达哥拉斯定理也称为勾股定理：直角三角形中，两条直角边的平方之和等于斜边的平方。

2.  常见勾股数：(3,4,5)、(5,12,13)、(7,24,25)、(8,15,17)、(9,40,41)

3.  额外勾股数：(11,60,61)、(12,35,37)、(13,84,85)、(15,112,113)

## 例题

**例题1**：在直角三角形中，两条直角边分别是3和4，求斜边的长度。

**解**：根据勾股定理：
斜边² = 3² + 4² = 9 + 16 = 25
斜边 = $\sqrt{25} = 5$

**例题2**：在直角三角形中，一条直角边是5，斜边是13，求另一条直角边的长度。

**解**：设另一条直角边为x，则：
$x^2 + 5^2 = 13^2$
$x^2 + 25 = 169$
$x^2 = 144$
$x = 12$

**例题3**：判断边长分别为6、8、10的三角形是否是直角三角形。

**解**：验证：$6^2 + 8^2 = 36 + 64 = 100 = 10^2$
所以是直角三角形（这是(3,4,5)勾股数的2倍）。

**例题4**：在30°-60°-90°的直角三角形中，30°角所对的直角边长为6，求斜边的长度。

**解**：30°角所对的直角边是斜边的一半
斜边 = 2 × 6 = 12

**例题5**：在45°-45°-90°的等腰直角三角形中，一条直角边长为5，求斜边的长度。

**解**：在45°-45°-90°的直角三角形中，斜边 = 直角边 × $\sqrt{2}$
斜边 = $5\sqrt{2}$

**例题6**：在直角三角形ABC中，∠C = 90°，AC = 8，BC = 6，求斜边AB上的中线长度。

**解**：根据勾股定理：$AB = \sqrt{8^2 + 6^2} = \sqrt{64 + 36} = \sqrt{100} = 10$
斜边上的中线 = $\frac{1}{2} \times AB = \frac{1}{2} \times 10 = 5$

**例题7**：在直角三角形中，两条直角边分别是a和b，斜边是c，内切圆半径为r，已知a = 6，b = 8，求r。

**解**：根据勾股定理：$c = \sqrt{6^2 + 8^2} = \sqrt{36 + 64} = 10$
根据公式：$r = \frac{a + b - c}{2} = \frac{6 + 8 - 10}{2} = \frac{4}{2} = 2$

**例题8**：在直角三角形ABC中，∠C = 90°，以AB、BC、CA为直径作半圆，已知AC = 3，BC = 4，求证：以AB为直径的半圆面积等于另外两个半圆面积之和。

**解**：根据勾股定理：$AB = \sqrt{3^2 + 4^2} = 5$
以AB为直径的半圆面积 = $\frac{1}{2} \times \pi \times (\frac{5}{2})^2 = \frac{25\pi}{8}$
以AC为直径的半圆面积 = $\frac{1}{2} \times \pi \times (\frac{3}{2})^2 = \frac{9\pi}{8}$
以BC为直径的半圆面积 = $\frac{1}{2} \times \pi \times (\frac{4}{2})^2 = \frac{16\pi}{8}$
$\frac{9\pi}{8} + \frac{16\pi}{8} = \frac{25\pi}{8}$ ✓
