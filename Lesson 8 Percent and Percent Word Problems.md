# Lesson 8 Percent and Percent Word Problems

# 第八讲 百分数与百分数应用题

# Definition of Percent 百分数的定义

1.  定义：

    百分比：是将数字与100进行比较的比值。符号是%。

2.  与其他数的转换：

<!-- -->

(1) 小数转百分数：乘以100%（如0.75 = 75%）。

(2) 分数转百分数：先化为小数再转换

# Basic types of percent problems 百分比问题的基本类型

1.  b的c%是多少：$\frac{x}{b} = \frac{c}{100}$，得出$x = \frac{c}{100}b$

2.  b的百分之多少是a：$\frac{a}{b} = \frac{x}{100}$，得出$a = \frac{x}{100}b$

3.  多少的c%是a：$\frac{a}{x} = \frac{c}{100}$，得出$a = \frac{c}{100}x$

# Percent of change 变化的百分比

一个数由a变到b，

1.  b\>a，即增长率：$\frac{b - a}{a} \times 100\%$

2.  a\>b，即减少率：$\frac{a - b}{a} \times 100\%$

## 例题

**例题1**：将0.65转换为百分数。

**解**：$0.65 = 0.65 \times 100\% = 65\%$

**例题2**：将 $\frac{3}{4}$ 转换为百分数。

**解**：$\frac{3}{4} = 0.75 = 75\%$

**例题3**：80的25%是多少？

**解**：$80 \times 25\% = 80 \times 0.25 = 20$

**例题4**：60的百分之多少是15？

**解**：设百分比为x%，则：
$\frac{15}{60} = \frac{x}{100}$
$x = \frac{15 \times 100}{60} = 25$
所以是25%。

**例题5**：多少的30%是24？

**解**：设这个数为x，则：
$x \times 30\% = 24$
$x = \frac{24}{0.30} = 80$

**例题6**：一件商品原价200元，现价150元，求降价百分比。

**解**：减少率 = $\frac{200 - 150}{200} \times 100\% = \frac{50}{200} \times 100\% = 25\%$

**例题7**：某公司去年利润是50万元，今年利润是65万元，求增长率。

**解**：增长率 = $\frac{65 - 50}{50} \times 100\% = \frac{15}{50} \times 100\% = 30\%$

**例题8**：一本书打8折后售价是64元，求原价。

**解**：设原价为x元，则：
$x \times 80\% = 64$
$x = \frac{64}{0.8} = 80$ 元

**例题9**：某商品先涨价20%，再降价20%，求最终价格是原价的百分之几？

**解**：设原价为100元
涨价后：$100 \times (1 + 20\%) = 120$ 元
降价后：$120 \times (1 - 20\%) = 120 \times 0.8 = 96$ 元
最终价格是原价的 $\frac{96}{100} = 96\%$
