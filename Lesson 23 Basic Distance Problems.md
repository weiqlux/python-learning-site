# Lesson 23 Basic Distance Problems

# 第二十三讲 基础行程问题

**一、行程三要素:路程(S)、速度(V)、时间(T)**

速度×时间=路程，记作V×T=S

路程÷速度=时间，记作S÷V=T

路程÷时间=速度，记作S÷T=V

1.  相遇问题:路程和=速度和×相遇时间

2.  追及问题:路程差=速度差×追及时间

3.  平均速度:平均速度=总路程÷总时间

## 例题

**例题1**：小明以每小时5千米的速度步行，走了3小时，求他走了多少千米。

**解**：路程 = 速度 × 时间 = $5 \times 3 = 15$ 千米

**例题2**：一辆汽车行驶了240千米，用了4小时，求汽车的速度。

**解**：速度 = 路程 ÷ 时间 = $240 \div 4 = 60$ 千米/小时

**例题3**：甲乙两人从相距100千米的两地同时出发，相向而行。甲的速度是每小时30千米，乙的速度是每小时20千米，求他们相遇的时间。

**解**：相遇时间 = 路程和 ÷ 速度和 = $100 \div (30 + 20) = 100 \div 50 = 2$ 小时

**例题4**：甲、乙两人从同一起点出发，甲的速度是每小时15千米，乙的速度是每小时10千米。如果甲比乙早出发1小时，求乙追上甲需要多长时间。

**解**：甲先走的路程 = $15 \times 1 = 15$ 千米（路程差）
追及时间 = 路程差 ÷ 速度差 = $15 \div (15 - 10) = 15 \div 5 = 3$ 小时

**例题5**：小明从家到学校，去时速度为每小时6千米，返回时速度为每小时4千米，求往返的平均速度。

**解**：设家到学校的距离为S千米
去时时间 = $\frac{S}{6}$ 小时
返回时间 = $\frac{S}{4}$ 小时
总时间 = $\frac{S}{6} + \frac{S}{4} = \frac{2S + 3S}{12} = \frac{5S}{12}$ 小时
总路程 = $2S$ 千米
平均速度 = 总路程 ÷ 总时间 = $2S \div \frac{5S}{12} = 2S \times \frac{12}{5S} = \frac{24}{5} = 4.8$ 千米/小时

**例题6**：一列火车长200米，以每秒20米的速度通过一座长800米的桥，求火车完全通过桥需要多长时间。

**解**：火车完全通过桥需要行驶的路程 = 桥长 + 火车长 = $800 + 200 = 1000$ 米
时间 = 路程 ÷ 速度 = $1000 \div 20 = 50$ 秒

**例题7**：甲、乙两人在400米的环形跑道上，从同一起点同时出发，同向而行。甲的速度是每秒5米，乙的速度是每秒3米，求甲第一次追上乙需要多长时间。

**解**：甲第一次追上乙需要多跑一圈（400米）
追及时间 = 路程差 ÷ 速度差 = $400 \div (5 - 3) = 400 \div 2 = 200$ 秒
