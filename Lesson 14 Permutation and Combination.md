# Lesson 14 Permutation and Combination

# 第十四讲 排列组合

# 加法原理

1.  如果事件$E_{1}$可以以$n_{1}$种方式发生，事件$E_{2}$可以以$n_{2}$种方式发生，事件$E_{3}$可以以$n_{k}$种方式发生，并且如果任何事件$E_{1}$，$E_{2}$或$E_{k}$发生，则工作完成，那么总共可以完成该工作的方式有$n_{1} + n_{2} + \ldots + n_{k}$种。

# 乘法原理

1.  当一个任务由k个独立的部分组成时，如果第一个部分可以用$n_{1}$种方式完成，第二个部分可以用$n_{2}$种方式完成，以此类推到第k个部分，可以用$n_{k}$种方式完成，那么总的可能完成该任务的方式为它们的乘积：$n_{1} \times n_{2} \times \ldots \times n_{k}$

# 阶乘和排列

1.  一个正整数的阶乘（factorial）是所有小于及等于该数的正整数的积，自然数n的阶乘写作n!。

2.  n! = 1\*2\*3\*4......\*n = n\*（n-1）\*（n-2）......\*1

    注意： 0！= 1

3.  如果该连续相乘的最小数不是1：n\*（n-1）\*...\*（n-k+1）= n!/(n-k)!

4.  常见的排列：从n个不同的元素中，任取m(m≤n)个元素按照一定的顺序排成一列，叫做从n个不同元素中取出m个元素的一个排列。顺序影响结果。

5.  这种排列的可能性求解如下：这m个元素，分别有n, n-1，...,
    到n-m+1种选取方式，则其乘积为n\*(n-1)\*...\*(n-m+1)，即
    P(n,m)或$A_{n}^{m}$，它们是排列数，即代表从n个不同元素中取出m(m≤n)个元素的所有排列的个数。

# 直线排列（排队）的常见题型和技巧

1.  捆绑法

    题型样式： 一队人ABCDEF排队，其中A和B必须相邻。

    解题技巧：将需要相邻的元素捆绑在一起，当作一个元素，但是不要忘记捆绑在一起的元素，内部也有顺序。

2.  插板法

    题型样式： 一队人ABCDEF排队，其中A和B不得相邻。

    解题技巧：先排没有不相邻限制的元素，再将不相邻的元素插入其中。

# 圆排列

1.  如果是n个人进行圆排列，那么结果为(n-1)!，即(人数-1)!

# 组合

1.  对r个元素进行排列，如果考虑顺序，排列个数是P(r,r)；如果不考虑顺序，个数为1。那么已知在考虑顺序时，n个元素中选取r个考虑顺序是P(n,r)，此时如果选取的r个是无序的，只要我们把P(n,r)的结果中重复的部分P(r,r)除掉，就是组合，得到的结果是P(n,r)/P(r,r)=n!/((n-r)!(r!))，又可以写作组合数$C_{n}^{r}$。

2.  $C_{n}^{r} = \frac{n!}{r!(n - r)!} = \left( \frac{n}{r} \right)$

3.  $C_{n}^{r}$=$C_{n}^{n - r}$

4.  $\left( \frac{n}{0} \right)$=1，$\left( \frac{n}{1} \right)$=n，$\left( \frac{n}{n} \right)$=1，$\left( \frac{n}{r} \right)$=$\left( \frac{n}{n - r} \right)$

# 其他特殊题型和技巧

1.  含有相同元素的排列：大范围中元素有顺序（排列），小范围去顺序（组合）。

2.  分组排列：相同元素的升级版，先大范围排列，对大范围分组，再逐个去顺序；或者先分组，按组合来算。

3.  递增数和递减数：如果一个数字的每个数字都大于紧挨着它的左边的数字（如果有的话），那么这个数字叫做递增数，如1234；反之，则为递减数。

4.  涂色问题：先确定哪些区域不相邻，可以用一样的颜色。

## 例题

**例题1**：从5个人中选3个人排队，有多少种不同的排法？

**解**：这是排列问题，$P(5,3) = 5 \times 4 \times 3 = 60$ 种

**例题2**：从5个人中选3个人组成小组（不考虑顺序），有多少种不同的选法？

**解**：这是组合问题，$C(5,3) = \frac{5!}{3!(5-3)!} = \frac{5 \times 4 \times 3}{3 \times 2 \times 1} = 10$ 种

**例题3**：5个人排队，其中A和B必须相邻，有多少种排法？

**解**：使用捆绑法，将A和B看作一个整体，有4个元素排列：$4! = 24$
A和B内部有2种顺序：$2! = 2$
总排法 = $24 \times 2 = 48$ 种

**例题4**：5个人排队，其中A和B不能相邻，有多少种排法？

**解**：使用插板法
先排其他3个人：$3! = 6$
这3个人形成4个空位（包括两端），A和B插入其中2个空位：$P(4,2) = 4 \times 3 = 12$
总排法 = $6 \times 12 = 72$ 种

**例题5**：5个人围成一圈，有多少种不同的排法？

**解**：圆排列 = $(5-1)! = 4! = 24$ 种

**例题6**：从10个不同的球中选3个，有多少种选法？

**解**：$C(10,3) = \frac{10!}{3!(10-3)!} = \frac{10 \times 9 \times 8}{3 \times 2 \times 1} = 120$ 种

**例题7**：验证组合性质：$C(8,3) = C(8,5)$。

**解**：$C(8,3) = \frac{8!}{3!5!} = 56$
$C(8,5) = \frac{8!}{5!3!} = 56$
所以 $C(8,3) = C(8,5)$ ✓

**例题8**：用数字1、2、3、4、5可以组成多少个没有重复数字的三位数？

**解**：$P(5,3) = 5 \times 4 \times 3 = 60$ 个

**例题9**：从3个红球、2个蓝球中选2个球，有多少种选法？

**解**：分类讨论：
- 2个红球：$C(3,2) = 3$
- 2个蓝球：$C(2,2) = 1$
- 1红1蓝：$C(3,1) \times C(2,1) = 3 \times 2 = 6$
总选法 = $3 + 1 + 6 = 10$ 种

**例题10**：用加法原理：从A地到B地有3条路，从B地到C地有4条路，从A地到C地有多少种走法？

**解**：根据乘法原理：$3 \times 4 = 12$ 种走法
