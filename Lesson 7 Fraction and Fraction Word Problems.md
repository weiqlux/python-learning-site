# Lesson 7 Fraction and Fraction Word Problems

# 第七讲 分数与分数应用题

# Classification of Fractions 分数的分类

1.  定义：

<!-- -->

(1) 分数
    fraction：分数表示一个整体被分成若干等份中的一部分，或者两个数的商，形式为$\frac{a}{b}$，其中a和b都是整数，$b \neq 0$。

(2) 真分数 proper fraction：分子小于分母，分数值小于1。

(3) 假分数 improper fraction：分子大于等于分母，分数值大于或等于1。

(4) 带分数 mixed number：有整数部分和分数部分，可写成假分数。

# Properties 性质

1.  等值分数 equivalent fraction：

    如果两个分数表示整体的同一部分，那么这两个分数相等。

2.  分数的基本定理 Fundamental Law of Fraction：

    对于任何分数$\frac{a}{b}$和任何数字$c \neq 0$，有$\frac{a}{b} = \frac{a \times c}{b \times c}$。

3.  最简分数：

    一个分数的分子和分母除了1以外没有公因数。最简分数通过把分子和分母所有的公因数约掉而得到。

4.  分数加减法法则：

<!-- -->

(1) 同分母分数加减，分母不变，分子相加减。

(2) 分数分母不同，先通分，分子再相加减。

<!-- -->

5.  分数乘法法则：分子的乘积做分子，分母的乘积做分母。

6.  分数除法法则：除以一个分数，等于乘以这个分数的倒数。

# Problem Solving Skills 解题技巧

1.  比较分数的大小：

<!-- -->

(1) 分母相同，分子大的分数大，分子小的分数小。

(2) 分子相同，分母大的分数小，分母小的分数大。

(3) 分子分母都不同，对角相乘。

<!-- -->

2.  连续分数求和常用公式：

<!-- -->

(1) $\frac{1}{n(n + 1)} = \frac{1}{n} - \frac{1}{n + 1}$

(2) $\frac{1}{n} = \frac{1}{2n} + \frac{1}{2n}$

(3) $\frac{1}{n(n + k)} = \frac{1}{k}(\frac{1}{n} - \frac{1}{n + k})$

<!-- -->

3.  连锁约分公式：

<!-- -->

(1) $\frac{1}{2} \times \frac{2}{3} \times \frac{3}{4} \times \ldots \times \frac{n}{n + 1} = \frac{1}{n + 1}$

(2) $(1 - \frac{1}{2}) \times (1 - \frac{1}{3}) \times (1 - \frac{1}{4}) \times \ldots \times (1 - \frac{1}{n + 1}) = \frac{1}{n + 1}$

(3) $\frac{3}{2} \times \frac{4}{3} \times \frac{5}{4} \times \ldots \times \frac{n + 1}{n} = \frac{n + 1}{2}$

(4) $(1 + \frac{1}{2}) \times (1 + \frac{1}{3}) \times (1 + \frac{1}{4}) \times \ldots \times (1 + \frac{1}{n}) = \frac{n + 1}{2}$

# Skills for Fraction Word Problems 分数应用题常用技巧

1.  A是B的几分之几：$A \div B = \frac{A}{B}$

2.  A的几分之几是B：$A \times (\ \ \ \ ) = B$ 或$B \div (\ \ \ \ ) = A$

3.  A比B多几分之几：$(A - B) \div B = \frac{A - B}{B}$
    或$A = B \times \lbrack 1 + (\ \ \ \ )\rbrack$

## 例题

**例题1**：将 $\frac{12}{18}$ 化为最简分数。

**解**：$\frac{12}{18} = \frac{12 \div 6}{18 \div 6} = \frac{2}{3}$

**例题2**：计算 $\frac{2}{3} + \frac{1}{4}$。

**解**：通分：$\frac{2}{3} = \frac{8}{12}$，$\frac{1}{4} = \frac{3}{12}$
$\frac{2}{3} + \frac{1}{4} = \frac{8}{12} + \frac{3}{12} = \frac{11}{12}$

**例题3**：计算 $\frac{5}{6} \times \frac{3}{10}$。

**解**：$\frac{5}{6} \times \frac{3}{10} = \frac{5 \times 3}{6 \times 10} = \frac{15}{60} = \frac{1}{4}$

**例题4**：计算 $\frac{2}{3} \div \frac{4}{5}$。

**解**：$\frac{2}{3} \div \frac{4}{5} = \frac{2}{3} \times \frac{5}{4} = \frac{10}{12} = \frac{5}{6}$

**例题5**：比较 $\frac{3}{4}$ 和 $\frac{5}{7}$ 的大小。

**解**：对角相乘：$3 \times 7 = 21$，$5 \times 4 = 20$
因为 $21 > 20$，所以 $\frac{3}{4} > \frac{5}{7}$

**例题6**：计算 $\frac{1}{1 \times 2} + \frac{1}{2 \times 3} + \frac{1}{3 \times 4} + \ldots + \frac{1}{9 \times 10}$。

**解**：利用公式 $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$
原式 = $(\frac{1}{1} - \frac{1}{2}) + (\frac{1}{2} - \frac{1}{3}) + (\frac{1}{3} - \frac{1}{4}) + \ldots + (\frac{1}{9} - \frac{1}{10})$
= $1 - \frac{1}{10} = \frac{9}{10}$

**例题7**：计算 $\frac{1}{2} \times \frac{2}{3} \times \frac{3}{4} \times \ldots \times \frac{9}{10}$。

**解**：利用连锁约分公式：
原式 = $\frac{1}{10}$

**例题8**：一本书有120页，小明读了 $\frac{3}{5}$，还剩多少页？

**解**：已读页数 = $120 \times \frac{3}{5} = 72$ 页
还剩页数 = $120 - 72 = 48$ 页

**例题9**：一根绳子长 $\frac{5}{6}$ 米，用去 $\frac{2}{3}$ 米，还剩多少米？

**解**：还剩 = $\frac{5}{6} - \frac{2}{3} = \frac{5}{6} - \frac{4}{6} = \frac{1}{6}$ 米

**例题10**：甲数是乙数的 $\frac{3}{4}$，乙数是丙数的 $\frac{2}{3}$，求甲数是丙数的几分之几？

**解**：设丙数为1，则乙数 = $\frac{2}{3}$，甲数 = $\frac{2}{3} \times \frac{3}{4} = \frac{1}{2}$
所以甲数是丙数的 $\frac{1}{2}$。
