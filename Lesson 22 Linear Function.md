**一、Coordinate and quadrant 坐标与象限**

1.  A plane Cartesian coordinate system 平面直角坐标系：

    在平面内，两条互相垂直且有公共原点的数轴，组成平面直角坐标系。其中，水平的数轴叫做x轴或横轴，取向右为正方向:铅直的数轴叫做v轴或纵轴，取向上为正方向:x轴和v轴统称坐标轴。它们的公共原点0称为直角坐标系的原点:建立了直角坐标系的平面，叫做坐标平面。

    ![1754034005647](media/image4.png){width="2.5520833333333335in"
    height="2.28125in"}

2.  Quadrant象限:

    为了便于描述坐标平面内点的位置，把坐标平面被x轴和y轴分割而成的四个部分，分别叫做第一象限、第二象限、第三象限、第四象限。

    *[注:x轴和y轴上的点(坐标轴上的点)，不属于任何一个象限。]{.underline}*

3.  The coordinates of the point 点的坐标:

    对于平面内任意一点P，过点P分别向x轴、y轴作垂线，垂足在x轴、y轴上对应的数a，b分别叫做点P的横坐标、纵坐标，有序数对(a，b)叫做点P的坐标。当a≠
    b时，(a，b)(b，a)是两个不同点的坐标。

2.  **Function and Linear function 函数与一次函数**

<!-- -->

1.  Definition of function 函数的定义：

    在一个变化过程中，有两个变量x和y，对于给定的每一个x值，y都有唯一确定的值与其对应，那么我们说x是自变量，y是因变量，y是x的函数。

2.  Linear function and its properties 一次函数的定义：

    (1)一次函数定义:如果$y = kx + b$($k,b$为常数，且$k \neq 0$)，那么，$y$叫做$x$的一次函数。当$b = 0,k \neq 0$时，$y$叫做$x$的正比例函数。

    (2)一次函数的图像:正比例函数$y = kx(k \neq 0)$的图像是经过原点(0，0)和点(1，k)的一条直线;一次函数$y = kx + b(k \neq 0)$是经过$(0，b)$、$( - \frac{b}{k}，0)$两点的一条直线。

    (3)一次函数的性质:正比例函数与一次函数的性质相同。当$k > 0$时，$y$随自

    变量$x$的增大而增大:当$k < 0$时，$y$随$x$的增大而减小。

## 例题

**例题1**：判断点A(3, 4)在哪个象限。

**解**：A的横坐标3 > 0，纵坐标4 > 0，所以点A在第一象限。

**例题2**：在平面直角坐标系中，点B(-2, 5)在哪个象限？

**解**：B的横坐标-2 < 0，纵坐标5 > 0，所以点B在第二象限。

**例题3**：判断 $y = 2x + 3$ 是否是一次函数。

**解**：是。符合 $y = kx + b$ 的形式，其中 $k = 2 \neq 0$，$b = 3$。

**例题4**：判断 $y = 5x$ 是否是正比例函数。

**解**：是。符合 $y = kx$ 的形式，其中 $k = 5 \neq 0$，$b = 0$。

**例题5**：一次函数 $y = 2x - 1$ 的图像经过哪些点？

**解**：当 $x = 0$ 时，$y = -1$，经过点(0, -1)
当 $y = 0$ 时，$0 = 2x - 1$，$x = \frac{1}{2}$，经过点($\frac{1}{2}$, 0)
所以图像经过点(0, -1)和($\frac{1}{2}$, 0)。

**例题6**：判断一次函数 $y = -3x + 2$ 的增减性。

**解**：因为 $k = -3 < 0$，所以y随x的增大而减小。

**例题7**：已知一次函数 $y = kx + b$ 经过点(1, 3)和(2, 5)，求k和b的值。

**解**：将点(1, 3)代入：$3 = k \times 1 + b$，即 $k + b = 3$
将点(2, 5)代入：$5 = k \times 2 + b$，即 $2k + b = 5$
解方程组：
$2k + b - (k + b) = 5 - 3$
$k = 2$
$b = 3 - k = 3 - 2 = 1$
所以 $k = 2$，$b = 1$，函数为 $y = 2x + 1$。

**例题8**：一次函数 $y = 4x - 2$ 与x轴、y轴的交点坐标分别是什么？

**解**：与x轴交点：令 $y = 0$，$0 = 4x - 2$，$x = \frac{1}{2}$，交点为($\frac{1}{2}$, 0)
与y轴交点：令 $x = 0$，$y = -2$，交点为(0, -2)
