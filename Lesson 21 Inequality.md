# 第二十一讲 不等式

**一、inequality and Linear inequality with one unknown number
不等式和含一个未知数的线性不等式**

1.  The concept of inequality 不等式的概念：

一般地，用符号"\<"(或"≤")"≠"连接的式子叫做不等式。"\>"(或"\>")

2.  Linear inequality with one unknown number 含一个未知数的线性不等式：

不等式中只含有一个未知数，且未知数的最高次数(指数)是"1"，如:x-5\<2.

3.  The solution of inequality 不等式的解：

使不等式成立的未知数的值，叫做不等式的解。

4.  The solution set of inequality 不等式的解集：

所有使不等式成立的未知数的值的集合，叫做不等式的解集。

5.  The properties of inequality 不等式的性质：

  ----------------------- ---------------------------------------------------------------- ---------------------------------------------------------
                          理论依据                                                         式子表达

  性质1                   不等式的两边同时加上(或减去)同一个数(或式子)，不等号的方向不变   若a\>b，则a±c\>b±c

  性质2                   不等式两边同时乘以(或除以)同一个正数，不等号的方向不变           若a\>b，c\>0，则ac\>bc或$\frac{a}{c}\  > \ \frac{b}{c}$

  性质3                   不等式两边同时乘以(或除以)同一个负数，不等号的方向改变           若a\>b，c\>0，则ac\<bc或$\frac{a}{c}\  < \ \frac{b}{c}$
  ----------------------- ---------------------------------------------------------------- ---------------------------------------------------------

6.  The method of solving Linear inequality with one unknown number
    解含一个未知数的线性不等式的方法：

    (1)去分母:根据不等式的基本性质，把不等式的两边同时乘以各分母的最小公倍数，得到整数系数的等式。

    (2)去括号:根据上括号的法则，特别要注意括号外面是负号时，去掉括号和负号，括号里面的各项要改变符号。

    (3)移项:根据不等式基本性质，一般把含有未知数的项移到不等式的左边，常数项移到不等式的右边。

    (4)合并同类项。把同类项进行合并计算。

    (5)将未知数的系数化为1:根据不等式基本性质，特别要注意系数化为1时，系数是负数，不等号要改变方向。

## 例题

**例题1**：解不等式 $3x - 5 > 7$。

**解**：$3x - 5 > 7$
$3x > 7 + 5$
$3x > 12$
$x > 4$

**例题2**：解不等式 $\frac{x}{2} + 3 \leq 5$。

**解**：$\frac{x}{2} + 3 \leq 5$
$\frac{x}{2} \leq 5 - 3$
$\frac{x}{2} \leq 2$
$x \leq 4$

**例题3**：解不等式 $-2x + 6 > 10$。

**解**：$-2x + 6 > 10$
$-2x > 10 - 6$
$-2x > 4$
$x < -2$（注意：除以负数，不等号方向改变）

**例题4**：解不等式组：
$$\begin{cases}
x + 3 > 5 \\
2x - 1 < 7
\end{cases}$$

**解**：解第一个不等式：$x + 3 > 5$，得 $x > 2$
解第二个不等式：$2x - 1 < 7$，得 $2x < 8$，即 $x < 4$
所以不等式组的解集为：$2 < x < 4$

**例题5**：解不等式组：
$$\begin{cases}
x - 2 \geq 0 \\
x + 1 < 4
\end{cases}$$

**解**：解第一个不等式：$x - 2 \geq 0$，得 $x \geq 2$
解第二个不等式：$x + 1 < 4$，得 $x < 3$
所以不等式组的解集为：$2 \leq x < 3$

**例题6**：解不等式 $2(x - 3) + 1 \geq 3x - 5$。

**解**：$2(x - 3) + 1 \geq 3x - 5$
$2x - 6 + 1 \geq 3x - 5$
$2x - 5 \geq 3x - 5$
$2x - 3x \geq -5 + 5$
$-x \geq 0$
$x \leq 0$

**例题7**：求不等式 $|x - 3| < 5$ 的解集。

**解**：$|x - 3| < 5$ 等价于 $-5 < x - 3 < 5$
即 $-5 + 3 < x < 5 + 3$
所以 $-2 < x < 8$
