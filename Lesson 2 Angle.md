# Classification of Angles 角的种类

1.  锐角 Acute Angle：在0°到90°之间的角。

2.  直角 Right Angle：等于90°的角。

3.  钝角 Obtuse Angle：在90°到180°之间的角。

4.  平角 Straight Angle：等于180°的角。

5.  对顶角的度数相等。

## 例题

**例题1**：判断下列角的类型：35°, 90°, 120°, 180°。

**解**：
- 35°：锐角（0° < 35° < 90°）
- 90°：直角
- 120°：钝角（90° < 120° < 180°）
- 180°：平角

**例题2**：两条直线相交，形成4个角。如果其中一个角是65°，求其他三个角的度数。

**解**：两条直线相交形成两对对顶角。
- 与65°对顶的角 = 65°
- 与65°相邻的两个角互补，每个角 = 180° - 65° = 115°
所以四个角分别为：65°, 115°, 65°, 115°

# Complementary Angles and Supplementary Angles 互余角和互补角

1.  互余角 complementary
    angles：如果两个锐角的度数之和为90°，那么这两个角互余。

2.  互补角 supplementary
    angles：如果两个角的度数之和为180°，那么这两个角互补。

## 例题

**例题1**：已知∠A和∠B互余，∠A = 37°，求∠B。

**解**：互余角之和 = 90°
∠B = 90° - ∠A = 90° - 37° = 53°

**例题2**：已知∠C和∠D互补，∠C = 125°，求∠D。

**解**：互补角之和 = 180°
∠D = 180° - ∠C = 180° - 125° = 55°

**例题3**：在三角形ABC中，∠A = 50°，∠B = 70°，求∠C。

**解**：三角形内角和 = 180°
∠C = 180° - ∠A - ∠B = 180° - 50° - 70° = 60°

**例题4**：求正八边形的内角和。

**解**：n边形内角和 = 180° × (n - 2)
正八边形内角和 = 180° × (8 - 2) = 180° × 6 = 1080°

**例题5**：一个多边形的内角和为900°，求这个多边形的边数。

**解**：设边数为n，则：
180° × (n - 2) = 900°
n - 2 = 5
n = 7
所以这是一个七边形。

2.  定理2：三角形的三个外角之和为360°。

## 例题

**例题1**：在等腰三角形ABC中，AB = AC，∠B = 40°，求∠A和∠C。

**解**：等腰三角形中，底角相等，所以∠B = ∠C = 40°
∠A = 180° - ∠B - ∠C = 180° - 40° - 40° = 100°

**例题2**：在等边三角形中，求每个内角的度数。

**解**：等边三角形三个角相等，每个角 = 180° ÷ 3 = 60°

**例题3**：在三角形ABC中，∠A = 50°，∠B = 60°，延长BC到D，求∠ACD（外角）。

**解**：根据外角定理，∠ACD = ∠A + ∠B = 50° + 60° = 110°

**例题4**：在三角形中，三个外角分别为100°, 120°, 140°，验证外角和定理。

**解**：三个外角之和 = 100° + 120° + 140° = 360°
验证了三角形的三个外角之和为360°。

## 例题

**例题1**：如图，直线l₁∥l₂，∠1 = 65°，求∠2、∠3、∠4的度数。

**解**：
- ∠2与∠1是同位角，所以∠2 = ∠1 = 65°
- ∠3与∠1是内错角，所以∠3 = ∠1 = 65°
- ∠4与∠1是同旁内角，所以∠4 = 180° - ∠1 = 180° - 65° = 115°

**例题2**：如图，AB∥CD，∠A = 110°，∠C = 70°，求∠E的度数。

**解**：过点E作EF∥AB，则EF∥CD
∠AEF = ∠A = 110°（内错角）
∠CEF = ∠C = 70°（内错角）
∠E = ∠AEF - ∠CEF = 110° - 70° = 40°

**例题3**：如图，l₁∥l₂，∠1 = 3x + 20°，∠2 = 2x + 40°，求x的值。

**解**：∠1与∠2是同旁内角，所以：
∠1 + ∠2 = 180°
(3x + 20°) + (2x + 40°) = 180°
5x + 60° = 180°
5x = 120°
x = 24°
