# Arithmetic Series 等差数列

1.  定义：前一项减后一项的差相等。

2.  五大要素：

    (1) 首项 first term：$a_{1}$

    (2) 末项 last term：$a_{n}$

    (3) 公差 common difference：d

    (4) 项数 number of terms：n

    (5) 和 sum：s

3.  求公差：

<!-- -->

(1) 公式： $d = a_{n + 1} - a_{n}$

    $$d = \frac{a_{m} - a_{n}}{m - n}$$

<!-- -->

4.  求和：

<!-- -->

(1) 公式（倒序相加法）： $s = (a_{1} + a_{n}) \times n \div 2$

(2) 前n个奇数的和：$1 + 3 + 5 + \ldots + (2n - 1) = n^{2}$

(3) 具有相同公差的整数数列求和（中位数法）：$s = m \times n$，其中m是序列中的中位数，n是参与求和的数字个数。

<!-- -->

5.  求项数：

<!-- -->

(1) 公式： $n = {(a}_{n} - a_{1}) \div d + 1$

<!-- -->

6.  求第n项：

<!-- -->

(1) 公式：

<!-- -->

1.  $a_{n} = a_{1} + (n - 1) \times d$

2.  $a_{n} = a_{m} + (n - m) \times d$

3.  $a_{n} = s_{n} - s_{n - 1}$

<!-- -->

(2) 如果m+n=p+q，那么$a_{m} + a_{n} = a_{p} + a_{q}$。

(3) 如果m+n=2q，那么$a_{m} + a_{n} = {2a}_{q}$。

## 例题

**例题1**：求等差数列 3, 7, 11, 15, ... 的第10项。

**解**：首项 $a_1 = 3$，公差 $d = 7 - 3 = 4$，第10项：
$$a_{10} = a_1 + (10 - 1) \times d = 3 + 9 \times 4 = 3 + 36 = 39$$

**例题2**：求等差数列 5, 12, 19, 26, ..., 89 的和。

**解**：首项 $a_1 = 5$，公差 $d = 12 - 5 = 7$，末项 $a_n = 89$。
先求项数：$n = \frac{a_n - a_1}{d} + 1 = \frac{89 - 5}{7} + 1 = \frac{84}{7} + 1 = 12 + 1 = 13$
再求和：$s = \frac{(a_1 + a_n) \times n}{2} = \frac{(5 + 89) \times 13}{2} = \frac{94 \times 13}{2} = 611$

**例题3**：求 $1 + 3 + 5 + 7 + \ldots + 99$ 的和。

**解**：这是前50个奇数的和，根据公式：
$$s = 50^2 = 2500$$

**例题4**：在等差数列中，已知 $a_3 = 10$，$a_7 = 22$，求 $a_{15}$。

**解**：由 $a_7 = a_3 + (7 - 3) \times d$，得：
$$22 = 10 + 4d$$
$$d = 3$$
所以 $a_{15} = a_7 + (15 - 7) \times d = 22 + 8 \times 3 = 22 + 24 = 46$

# Geometric Series 等比数列

1.  定义：前一项与后一项的比值相等。

2.  五大要素：

    (1) 首项 first term：$a_{1}$

    (2) 末项 last term：$a_{n}$

    (3) 公比 common ratio：q

    (4) 项数 number of terms：n

    (5) 和 sum：s

3.  求第n项：

<!-- -->

(1) 公式：

<!-- -->

1.  $a_{n} = a_{1} \times q^{n - 1}\ (a_{1}q \neq 0)$

2.  $a_{n} = a_{m} \times q^{n - m}\ (a_{1}q \neq 0)$

3.  ${a_{n}}^{2} = a_{n - k} \times a_{n + k}\ (n \geq k)$

<!-- -->

(2) 如果m+n=p+q，那么$a_{m} \times a_{n} = a_{p} \times a_{q}$。

(3) 如果m+n=2t，那么$a_{m} \times a_{n} = {a_{t}}^{2}$。

## 例题

**例题1**：求等比数列 2, 6, 18, 54, ... 的第8项。

**解**：首项 $a_1 = 2$，公比 $q = \frac{6}{2} = 3$，第8项：
$$a_8 = a_1 \times q^{8-1} = 2 \times 3^7 = 2 \times 2187 = 4374$$

**例题2**：在等比数列中，已知 $a_2 = 12$，$a_5 = 96$，求 $a_8$。

**解**：由 $a_5 = a_2 \times q^{5-2}$，得：
$$96 = 12 \times q^3$$
$$q^3 = 8$$
$$q = 2$$
所以 $a_8 = a_5 \times q^{8-5} = 96 \times 2^3 = 96 \times 8 = 768$

**例题3**：在等比数列中，已知 $a_3 = 4$，$a_7 = 64$，求 $a_5$。

**解**：由性质 ${a_5}^2 = a_3 \times a_7$，得：
$${a_5}^2 = 4 \times 64 = 256$$
$$a_5 = 16$$（因为 $a_5$ 在 $a_3$ 和 $a_7$ 之间，取正值）

**例题4**：求等比数列 1, 2, 4, 8, ..., 1024 的项数。

**解**：首项 $a_1 = 1$，公比 $q = 2$，末项 $a_n = 1024$。
由 $a_n = a_1 \times q^{n-1}$，得：
$$1024 = 1 \times 2^{n-1}$$
$$2^{n-1} = 1024 = 2^{10}$$
$$n - 1 = 10$$
$$n = 11$$

# Numerical Table 数表

1.  规律：

<!-- -->

(1) 按行或列排列的数字，通常有固定的增量模式。

(2) 通过行数和列数可以定位具体数字，或通过数字反推行列位置。

## 例题

**例题1**：观察下面的数表，找出第5行第3列的数。

```
1   2   3   4   5   ...
3   5   7   9   11  ...
5   8   11  14  17  ...
7   11  15  19  23  ...
9   14  19  24  29  ...
...
```

**解**：观察规律：
- 第1行：1, 2, 3, 4, 5, ...（公差为1）
- 第2行：3, 5, 7, 9, 11, ...（公差为2）
- 第3行：5, 8, 11, 14, 17, ...（公差为3）
- 第4行：7, 11, 15, 19, 23, ...（公差为4）
- 第5行：9, 14, 19, 24, 29, ...（公差为5）

第5行首项为9，公差为5，第3列：
$$a_3 = 9 + (3-1) \times 5 = 9 + 10 = 19$$

**例题2**：在下面的数表中，数字2024出现在第几行第几列？

```
1   4   7   10  13  ...
2   5   8   11  14  ...
3   6   9   12  15  ...
...
```

**解**：观察规律：
- 第1列：1, 2, 3, ...（公差为1）
- 第2列：4, 5, 6, ...（公差为1）
- 第3列：7, 8, 9, ...（公差为1）

每列都是等差数列，第n列首项为 $3n - 2$。
设2024在第n列第m行，则：
$$2024 = 3n - 2 + (m - 1) \times 1$$
$$2024 = 3n + m - 3$$
$$3n + m = 2027$$

由于 $2027 \div 3 = 675 \cdots 2$，所以 $n = 675$，$m = 2027 - 3 \times 675 = 2$。
验证：第675列首项为 $3 \times 675 - 2 = 2023$，第2项为 $2023 + 1 = 2024$。
所以2024出现在第2行第675列。

# Pascal's Triangle 杨辉三角

1.  规律：

<!-- -->

(1) 第n行个数 = n

(2) 前n行个数 $= 1 + 2 + 3 + \ldots + n$

(3) 第n行所有数之和 $= 2^{n - 1}$

(4) 第n行第二个数 = n-1

(5) 第n行第3个数 $= 1 + 2 + 3 + \ldots + (n - 2)$

## 例题

**例题1**：杨辉三角第10行有多少个数？前10行共有多少个数？

**解**：
- 第10行个数 = 10
- 前10行个数 = $1 + 2 + 3 + \ldots + 10 = \frac{10 \times 11}{2} = 55$

**例题2**：求杨辉三角第8行所有数的和。

**解**：根据公式，第n行所有数之和 = $2^{n-1}$，所以：
第8行所有数之和 = $2^{8-1} = 2^7 = 128$

**例题3**：求杨辉三角第12行第2个数和第3个数。

**解**：
- 第12行第2个数 = $12 - 1 = 11$
- 第12行第3个数 = $1 + 2 + 3 + \ldots + (12 - 2) = 1 + 2 + 3 + \ldots + 10 = \frac{10 \times 11}{2} = 55$

**例题4**：杨辉三角中，第n行的第k个数（从1开始计数）等于组合数 $C_{n-1}^{k-1}$。求第7行第4个数。

**解**：第7行第4个数 = $C_{7-1}^{4-1} = C_6^3 = \frac{6!}{3!(6-3)!} = \frac{6 \times 5 \times 4}{3 \times 2 \times 1} = 20$

或者直接计算：第7行第3个数 = $1 + 2 + 3 + 4 = 10$，第4个数 = $10 + 5 = 15$（需要观察杨辉三角的递推规律）

实际上，杨辉三角第7行为：1, 6, 15, 20, 15, 6, 1，所以第4个数是20。
